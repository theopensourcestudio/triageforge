# Support

TriageForge is maintained in public on GitHub.

- Ask usage questions in [GitHub Discussions](https://github.com/theopensourcestudio/triageforge/discussions).
- Report reproducible bugs through the [bug report form](https://github.com/theopensourcestudio/triageforge/issues/new?template=bug_report.yml).
- Propose features through the [feature request form](https://github.com/theopensourcestudio/triageforge/issues/new?template=feature_request.yml).
- Report security vulnerabilities privately using [GitHub Security Advisories](https://github.com/theopensourcestudio/triageforge/security/advisories/new).

Please do not disclose vulnerabilities in public issues.
